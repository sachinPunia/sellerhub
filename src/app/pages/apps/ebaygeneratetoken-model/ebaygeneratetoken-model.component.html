Generate a new eBay token?

You need to generate an eBay token in order to list your inventory into eBay using Sellershub selling tools. You can do this by login into eBay and follow the integration steps.This will take less than 1 minute to complete. If your login is successful, eBay will issue a token to Sellershub which will be used to interact with eBay.
If you do not have an account with eBay, you can create an account at www.ebay.com
Sellershub is an eBay Compatible Application.
